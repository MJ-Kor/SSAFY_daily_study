## Modular Arithmetic

> 개념

- $r = a \ \% \ m$
- $r = a \ mod \ m$
- a를 m으로 나누었을때의 나머지이다.
<br><br>

> Modular 연산의 특징

- 분배 법칙의 성립

1. $(A \ \pm \ B) \ mod \ C = (A \ mod \ C \pm \ B \ mod \ C) \ mod \ C$ 
2. $(A \ * \ B) \ mod \ C = (A \ mod \ C * \ B \ mod \ C) \ mod \ C$ 
<br><br>

> 문제 상황

- 어떠한 매우 큰 수의 연산을 진행하고, 그 수를 p로 나누었을 때의 나머지를 구해야 하는 상황
